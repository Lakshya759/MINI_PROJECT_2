# 📚 ShareSphere

**ShareSphere** is a MERN stack-based web application that enables users to upload, share, search, and download PDF files seamlessly. It provides a secure and collaborative platform for accessing community-shared documents.

---

## 🚀 Features

* 🔐 **User Authentication** (JWT-based login/signup)
* 📤 Upload PDF files
* 📥 Download PDFs shared by other users
* 🔎 **Search and filtering system** for easy document discovery
* 📂 Centralized document sharing system

---

## 🛠️ Tech Stack

### 🔹 Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MongoDB

### 🔹 Other Tools

* Git & GitHub
* AWS EC2 (for deployment)

---

## 🧩 System Architecture

* **Frontend (React):** Handles UI and user interactions
* **Backend (Node + Express):** Manages APIs, authentication, and file handling
* **Database (MongoDB):** Stores user data and PDF metadata

---

## 📌 How It Works

1. Users sign up or log in securely using JWT authentication.
2. Authenticated users can upload PDF files.
3. File metadata is stored in MongoDB.
4. Users can search and filter documents.
5. PDFs can be downloaded by other users.

## 📦 Installation & Setup

### 1️⃣ Clone the repository

```
https://github.com/Lakshya759/MINI_PROJECT_2
```

### 2️⃣ Navigate to project directory

```
cd sharesphere
```

### 3️⃣ Install dependencies

#### Backend

```
cd backend
npm install
```

#### Frontend

```
cd frontend
npm install
```

### 4️⃣ Run the application

#### Start backend

```
cd backend
npm start
```

#### Start frontend

```
cd frontend
npm start
```
